This '/source-sample' folder and all the files it contains are committed for doing a basic testing. Please keep it. This are the files that you would need for the "source" folder.

- /bpm-1 and /bpm-2 : a couple of monochrome icons in each folder.
- /objects-1 and /objects-2 : a couple of multicolor icons in each folder.
- my-color-states.json : a sample color states file. This is required for the processor to work.
  it details the colors for the different states, for monochrome icons, as wel as for multicolor.
- figma-exported-icons.zip : this zip has the icons inside /bpm and /objects. This zip was exported from figma.
- figma-sample-icons.fig : this figma contains the svg designs for the four sample icons.
