This sample-files folder and all the files it contains are being committed for doing a basic testing. Please keep it.
This folder contains all the files that you would need for the "source" folder.

- /bpm : a couple of monochrome icons
- /objects : a couple of multicolor icons
- color-states.json : a sample color states file. This is required for the processor to work.
  it details the colors for the different states, for monochrome icons, as wel as for multicolor.
- figma-exported-icons.zip : this zip has the icons inside /bpm and /objects. This zip was exported from figma.
- figma-sample-icons.fig : this figma contains the svg designs for the four sample icons.
