background

CSS utility classes for background colors offer a fast, consistent way to apply styles, reducing repetitive code. They speed up development and simplify maintenance by centralizing color updates, ensuring uniformity across the project.

Apply the .elevation-01 background color class only when the container has no ancestor with an elevation background. This approach maintains a clear visual hierarchy by enhancing contrast between nested elevation containers.

Apply the .elevation-02 background color class when the parent container's has a .elevation-01 background. This maintains a clear visual hierarchy by enhancing contrast between nested elevation containers.

Apply the .elevation-03 background color class when the parent container's has a .elevation-02 background. This maintains a clear visual hierarchy by enhancing contrast between nested elevation containers.
