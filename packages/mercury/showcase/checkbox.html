ch-checkbox

A checkbox is a form element that allows users to select one or more options from a set by checking. It is commonly used for tasks such as selecting preferences, or agreeing to terms and conditions, providing a clear and simple way for users to make multiple independent selections within a form.

enabled indeterminate

An indeterminate checkbox is a checkbox that is in a state that is neither checked nor unchecked. This state indicates that only some, but not all, sub-options related to the checkbox are selected.
